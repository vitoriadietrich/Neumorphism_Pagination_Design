# CSS3 - Neumorphism Pagination Design | CodingNepal

Um projeto simples de **paginação com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar uma **pagination moderna com estilo neumorphism**, utilizando sombras suaves para um visual elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Sistema de paginação
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
